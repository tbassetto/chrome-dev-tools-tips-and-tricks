Console API

  • console.log n'est pas seul :)
  • console.log accepte plusieurs arguments
  • console.log('Name: %s, Intergalactic Credits: %d.', 'Alex', -100)
// Name: Alex, Intergalactic Credits: -100.
  • console.dir
  • console.time et console.timeEnd
  • console.group et console.groupEnd
  • console.count
  • Exemples

Console

  • Peut-être appelé de n'impotre quel onglet via ESC
  • Auto-complétion
  • Historique via ↑ et ↓
  • $(id)
  • $$(selecteur)
  • $0 — $4 : historique des nœuds sélectionnés
  • clear()
  • copy()

Panneau CSS

  • Auto-complétion
  • Palette de couleur
  • Créer un nouveau sélecteur
  • Édition de :hover, :focus, etc.
  • Incrémenter les valeurs avec les flèches (par 0.1, 1, 10, 100 avec combinaisons de Shift, Alt, etc.)
  • Clic sur le nom du fichier CSS pour avoir accès au diff + historique
